# Hostel Gate Pass Management System (Frontend)

Modern React + Vite + Tailwind app with a royal 3D landing page (Three.js + GSAP) and role-specific modules.

## Environment
Create `.env`:
```
VITE_BACKEND_URL=http://localhost:8000
```

## Scripts
- dev: start Vite dev server
- build: production build
- preview: preview build

## Pages
- Landing (3D)
- Student: register, login, apply pass, send OTP
- Parent: verify OTP
- Warden: approve/reject
- Security: scan status
- Admin: view logs

## Notes
- Uses React Router 6
- Tailwind preconfigured
- Three.js and GSAP for hero animation
